Khan-Draw

Before you start playing, it's worth noting that Khan-Draw will only work with a Desktop browser using Chrome! That is:

Tablets and Mobile devices will not be able to draw properly.

Firefox and Internet Explorer can't (currently) record video.

Still, it's worth having a tinker!!

Khan-Draw

This is just a fun widget to tinker with newish technologies. The idea is to replicate the drawing software used on Khan Academy, except online. Remember kids, imitation is the sincerest form of flattery!

Made by oodavid.com

Drawing

Click and drag to make curves. With a caveat...

  • There is a bug in RecordRTC where the screen isn't properly recorded while the mouse is held down...
  • ...so I've added a workaround, simply hold the ALT key to draw; no need to mousedown!
  • BTW: The "smooth drawing" is based on this tutorial.

Color Selection

The logo shows the currently selected color, but there are a number of ways to change color

  • Click the color icon
  • Use the number keys 0-9
  • TAB and SHIFT+TAB to cycle

Recording a video

  • Click "record" to begin the recording and "stop" when your finished
  • It often takes a while to compile the final recording
  • I've used the RecordRTC project for this

Using a Tablet

  • I recommend mapping the following keys:
    • ALT - to the main pen button to start drawing
    • TAB - to the seondary pen button to cycle colors
  • Once the pointer API is more widely supported we'll be able to use the pen pressure and tilt to make more fluid drawings using the tablet.
